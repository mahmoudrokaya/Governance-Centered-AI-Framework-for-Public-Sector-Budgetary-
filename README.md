# Governance-Centered AI Framework for Public-Sector Budgetary Analysis

## Overview

This repository contains the complete code, data-processing pipeline, experimental framework, and reproducibility resources developed for the study:

**Transforming AI-Driven Solutions for Public Financial Governance and Budget Performance Analysis**

The project proposes a governance-centered artificial intelligence framework for public-sector budgetary analysis and fiscal anomaly detection. The framework integrates governance-oriented indicators with machine learning models to support transparent, auditable, and interpretable financial governance assessment.

---

## Research Objectives

The framework was developed to address key challenges in AI-driven public financial governance:

* Fiscal anomaly detection
* Budget performance analysis
* Governance metric construction and validation
* Model transparency and auditability
* Reproducible public-sector AI research

## Experimental Pipeline

### Stage 1 — Data Inventory and Audit

* Audit of available Ministry of Finance data
* Data quality assessment
* Dataset expansion planning

### Stage 2 — Data Extraction and Standardization

* Extraction of budget indicators
* Standardization of fiscal categories
* Construction of analytical datasets

### Stage 3 — Panel Dataset Construction

* Multi-period fiscal panel generation
* Feature engineering
* Entity-level aggregation

### Stage 4 — Governance Metric Development

The framework defines governance-oriented indicators including:

* Governance Alignment Index
* Traceability Index
* Interpretability Index
* Audit Readiness Index
* Risk Signal Completeness Index
* Temporal Support Index
* Decision Usability Index

### Stage 5 — AI Modeling

Three modeling strategies are evaluated:

1. Conventional Statistical Baseline
2. Standalone AI Model
3. Governance-Centered AI Model

### Stage 6 — Governance Validation Studies

#### Study A

Independent predictive value analysis.

#### Study B

Leave-one-metric-out ablation analysis.

#### Study C

Statistical validation of governance metrics using:

* Mann–Whitney U tests
* Welch t-tests
* Spearman correlations
* Point-biserial correlations
* Mutual information
* Cohen’s d
* Hedges’ g
* Cliff’s Delta
* Bootstrap confidence intervals
